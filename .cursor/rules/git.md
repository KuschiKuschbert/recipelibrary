# Git — branches and commits

- **Branches:** `improvement/…`, `fix/…`, `chore/…`, `docs/…`. Avoid unscoped work directly on `main` for multi-file features.
- **Commits:** [Conventional Commits](https://www.conventionalcommits.org/) — `feat(html):`, `fix(riviera):`, `chore(cursor):`, `docs:`.
- **Public repo:** Never commit secrets, API keys, or machine-only paths.
