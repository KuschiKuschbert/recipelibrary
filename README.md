## Files

### `claude_index/` — Claude Project Knowledge indexes
15 compact JSON files (~500KB each), optimised to fit within Claude's 200K token context window.
Use these to search the full library from any Claude project or device.

**Schema per recipe entry:**
```json
{
  "id": "recipe-slug",
  "name": "Recipe Name",
  "cat": "Meat & Poultry",
  "cui": "Greek",
  "protein": ["chicken"],
  "tags": ["gluten-free", "high-protein"],
  "ing": ["Chicken thigh", "Lemon", "Garlic"]
}
```
